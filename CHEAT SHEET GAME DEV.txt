[CREATE GITHUB ACCOUNT]

= Create GitHub account
= Create new repository
= Set to public, set ignore template to unity, check README file
= Yung location ng github file mo ay nakadepend kung saan mo ilalagay yung github folder sa sarili mong computer
= Go to desktop.github.com to install in desktop
= after install, search github on search bar sa windows
= connect your github account, then authorize this desktop para ma-edit mo sa sariling computer
= check your repository folder sa file explorer (nakadepend ito kung saang location mo nilagay kanina sa 'create new repository')

[CONNECT GITHUB TO UNITY]
= Open Unity Hub
= Create New Project
= 2D, file location is same as the repository location
= Go to your repository folder in file explorer
= open it and double click .gitignore folder
= vscode will pop out, then remove the first slash from line 5 to 11
= Ctrl+S to save, then close
= check your GitHub desktop, look on the left column, (dapat from 20,000+ files marereduce to '32' changed files)

[COMMIT GITHUB TO BROSWER GITHUB]
= On GitHub desktop, left column sa may baba meron 'Summary'
= Type mo sa summary anything/ summary ng gawa mo
= Click "commit to main" button
= Click "push origin" button
= Antayin matapos yung push origin
= Check GitHub browser kung naupload doon (refresh browser)

[INVITE MGA KAGRUPO/SHARED WORK SA GITHUB]
*Kung ikaw mag-iinvite
= On GitHub browser >> Go to Settings >> Collaborators >> Add People
*Kung ikaw inimbitahan
= Check notifications kung na-add

[UNITY GAME HANDLER/MANAGER]
= On Unity Hub >> Game Object >> Create Empty
= Name it "Game Handler'
= Right column >> Add Component >> New Script >> Name it to "Game_Handler"
= Sa bottom (Assets Panel) double-click to open VS Code
= *Enter C# code

[UNITY TEXT]
= GameObjects >> UI >> Text-MeshPro >> Import TMP Essentials
= *For the meantime wag muna galawin Canvas & Event system sa left corner
= Right column >> New Text (dito mo i-rereplace yung laman ng text box)
= Long press & drag Text (TMP) into the scene
= Zoom-out to see the text on screen
= Go to Game >> Free Aspect >> Change into WXGA resolution
= Go to Game Handler
= Drag Text (TMP) to Timer GO in right column
